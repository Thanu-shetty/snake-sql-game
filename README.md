## How to Play

1. Use arrow keys to control the snake
2. Reach gray food items to trigger SQL challenges
3. Answer SQL questions correctly to eat the food
4. Each correct answer increases your score
5. Avoid hitting walls or yourself

## Technologies Used

- **Backend**: Python, Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite
- **Game**: Canvas API
