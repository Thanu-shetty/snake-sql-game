# Snake SQL Game

A fun educational game that combines classic Snake gameplay with SQL query challenges.

## Features
- Classic Snake gameplay with arrow keys
- SQL query challenges when eating special food
- Score tracking and level progression
- Built with Flask, HTML5 Canvas, and JavaScript

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run: `python app.py`
3. Open http://localhost:5000
